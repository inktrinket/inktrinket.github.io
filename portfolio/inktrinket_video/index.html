"inktrinket.video" Web Project

"inktrinket.video" is my website! It was created over the span of 3 to 4 months in collaboration with Parker Lippstock, with everything you see here fully handmade from scratch - from the graphics to the code of the website itself. We went through multiple design iterations that were tested day and night before we ended up with the version you see here today! Lippstock was in charge of constructing a majority of the technical elements of the website, such as the code and graphics - I was the project lead, responsible for visualizing the several concept iterations, creating the brand identity/philosophy, and getting the website to where it needed to go.
